# IVT

圖生影片提示詞說明指南。

這個倉庫目前提供一個可直接開啟的單檔 HTML 指南，整理了圖生影片常用的提示詞寫法，包含：

- 運鏡與鏡頭語言
- 人物動作與表情變化
- 光線、天氣、色彩與攝影風格
- 台詞、聲音、環境音與配樂
- 常見錯誤與替代寫法
- 可直接套用的英文 Prompt 與繁體中文釋義

## 使用方式

直接在瀏覽器開啟：

```bash
open image-to-video-prompt-guide.html
```

如果你是在沒有 `open` 指令的環境，也可以直接雙擊檔案，或用任意瀏覽器打開。

如果要作為 GitHub Pages 首頁使用，根目錄的 [index.html](./index.html) 已經提供入口頁。

## 內容特色

- 單檔 HTML，便於保存、分享與二次修改
- 英文提示詞旁附繁體中文對照
- 包含模板、詞庫、示例與實務整理
- 以官方文件為主要整理依據，再做跨模型通用化歸納
